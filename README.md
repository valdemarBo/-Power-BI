# Power BI Dashboard

## Описание проекта
Данный проект представляет собой дашборд, разработанный в **Microsoft Power BI**, для анализа данных из файла `База данных.xlsx`. Визуализация построена в соответствии с требованиями, описанными в техническом задании (`ТЗ.docx`).

Основная цель — предоставить наглядную аналитику и ключевые метрики для поддержки принятия решений.

## Содержимое репозитория
- **`bolshakov_1stage.pbix`** — основной файл отчета Power BI.
- **`База данных.xlsx`** — исходные данные (Excel таблица).
- **`ТЗ.docx`** — техническое задание на разработку отчета.

## Требования для работы
Для открытия и просмотра файла `.pbix` необходимо установить:
- **Microsoft Power BI Desktop** (бесплатная версия).
  - [Скачать с официального сайта](https://powerbi.microsoft.com/ru-ru/desktop/)
- **Microsoft Excel** (для просмотра исходных данных, опционально).
- **Microsoft Word** (для чтения ТЗ, опционально).

> ⚠️ **Важно:** Для корректного обновления данных в `.pbix` файл `База данных.xlsx` должен находиться в той же папке, что и сам отчет, либо путь к нему нужно будет обновить вручную в Power BI (меню *Преобразовать данные* -> *Источники данных*).

## Как использовать
1. Клонируйте репозиторий или скачайте все файлы в одну папку.
2. Убедитесь, что `bolshakov_1stage.pbix` и `База данных.xlsx` лежат в одном каталоге.
3. Откройте `bolshakov_1stage.pbix` с помощью Power BI Desktop.
4. При необходимости нажмите «Обновить» (Refresh), чтобы загрузить актуальные данные из Excel.
